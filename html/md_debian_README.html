README for Debian build of BES

Using make:

To generate a debian package using make type the command:

1 make deb

Using the command line:

To generate the debian package using command line commands:

1 make dist
2 mv bes-<PACKAGE-PACKAGE_VERSION>.tar.gz ..
3 cd ..
4 mv bes-<PACKAGE-PACKAGE_VERSION>.tar.gz bes_<PACKAGE-PACKAGE_VERSION>.orig.tar.gz
5 tar xzf bes_<PACKAGE-PACKAGE_VERSION>.orig.tar.gz
6 cd bes-<PACKAGE-PACKAGE_VERSION>/
7 cp -r ../bes/debian/ .
8 debuild -us -uc
