std::vector< std::vector< string > > StringMatch::match ( const char *  regex,

std::vector< size_t > &  groups,

)
static

Given a 'row' this function extract all the matching groups (listed in groups) of the 'regex'. This is done while regex match substrings in the row string. The result is a vector containing vectors of strings matched by groups. 'row' is modified eliminating all the matched regex.
Note: that the size of the inner vector is the same as the 'groups' and the size of the outer vector is the number of matching string (in the row).

Deprecated:
this function is deprecated and is going to be removed. Use the safer and faster:
list<string[]> match(regex,groups,row)
