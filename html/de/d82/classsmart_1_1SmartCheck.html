Detailed Description

Class which implements methods to check STATICALLY (at compile time) the template parameter T (and K for maps). This is useful to specialize actions if functions and constructors which use template should know if the passed object is a pointer or not (for example). Supported checks are applied on a passed object of T{,K} type:

  • Is T a pointer
  • Is T a null pointer ()
  • Is T{,K} a Structure (list,vector,map)
  • Is T an array[]
