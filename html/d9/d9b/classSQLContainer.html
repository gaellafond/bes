Detailed Description

Abstract class which defines methods used by the SQLHandler to explore and get data from a dataset. Implements Clone to be usable with std::shared_ptr<>.

smart::Clone BESFileContainer


A SQLContainer implementation MUST:

  • Define a structure matching SQLContainer methods
    See also
    SQLTextDefinition
  • Define a persistence storage type such as TXT,XML,BINARY,...
  • Use relatives API to implement the virtual methods follow comments for instructions


Note
: Since SQLContainer is basically a BESFileContainer it implements methods to check the age of a loaded Container (looking at the file modification time date).
See also
SQLRequestHandler
SQLPlugin
SQLStorageContainerVolatile
Note
: The desired SQLContainer implementation to load is chosen by the SELECT_CONTAINER function of the SQLContainerFactory.
See also
SQLContainerFactory

virtual bool SQLContainer::setNext ( )
pure virtual

returns true if there is next section in the list and set actual section to the next one.
In a list from 0 to (n-1)

  • reset() put cursor to 0
  • FIRST setNext() call set cursor to 1
  • LAST setNext() call set cursor to n-1 returning true
  • ALL the successive calls to the setNext() must return false until reset() is call
