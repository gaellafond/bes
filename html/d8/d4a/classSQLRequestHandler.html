Detailed Description

SQLHandler base RequestHandler which is the main SQL request handler.
This RequestHandler act as a wrapper for all the SQL type requests.
It uses its functions to search and run loaded SQLPlugin which effectively implements Components and methods to build DDS, DAS, DATA ... objects.
