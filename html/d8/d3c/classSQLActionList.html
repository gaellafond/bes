Detailed Description

template<class ARGS_TYPE, class OUT_TYPE = void>
class SQLActionList< ARGS_TYPE, OUT_TYPE >

Interface which defines methods to implement to build a working ActionList. An ActionList is a collection of objects which implements a method defined as: OUT_TYPE * action(ARGS_TYPE*)
Note: This is typically defined in a SQLAction. This interface defines typical methods of a list such as:

  • hasNext
  • getSize
  • reset
    define also a doNext method instead of a getNext to free the implementing class to define its own action() storage type.
