Detailed Description

template<class JOIN = void>
class SQLErrorManager< JOIN >

SPECIALIZATION Implements an ActionManager to build an ErrorManager which fix.

  • JOIN==MERGE==OUT_TYPE so the returning type (OUT_TYPE) will be the same type of the SQLAction composing SQLActionList managed by this SQLActionManager.

Execute actions generated by the ActionFactory and stores results into the JOIN status variable. CODE_TYPE the type of the code used by the passed ActionFactory ARGS_TYPE the type of the argument used by the actions executed by this ActionFactory OUT_TYPE the output type of executed action.

Parameters
_afThe ActionFactory to be executed
merge_fThe function used to merge OUT_TYPE in MERGE_TYPE
free_mergedA boolean indicating if merged elements (OUT_TYPE) must be deleted (true) or not (false)
join_fThe function used to join MERGE_TYPE in JOIN_TYPE
multiple_argsA boolean indicating if getArgs should be called for each Action of the ActionList (true) or only one time (false) as for getCode. This mean that ALL the action in the selected ActionList will run using same argument.
