Detailed Description

template<class K, class V, class T = std::shared_ptr<V>>
class smart::SmartValueMap< K, V, T >

This map stores object using smart pointers. Note that keys are NOT stored as smart object. This speed up find searches and the resulting size of the map is smaller than the SmartMap one.

CLONE is set to 1 smart::Clone
