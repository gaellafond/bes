Detailed Description

template<class IN, class OUT>
class SQLObjectAction< IN, OUT >

class which extends an SQLAction to implement a stateful action. The state variable is a pointer which store the result of the action until freeValue() is called or the class dtor is called.

Note
: implementing the action you have to set the buffer calling setVal(IN*) method before return.
: CLONE is re-implemented to keep the status of this SQLAction
