template<class SQL_TYPE, class ODBC_TYPE>
virtual ODBC_TYPE* SQLTypeConnector< SQL_TYPE, ODBC_TYPE >::getNext ( size_t  next = 1)
pure virtual

Returns a value of ODBC_TYPE type containing the next value in this result set.
It is pointed by the ACTUAL position:

  • column -> getCol()
  • row -> getRow()
    In a Sequence the NEXT value should be:
  • The NEXT COLUMN value of the actual row
    OR
  • The first value of the NEXT ROW
    Note
    This method should also update cursors to the next position using setNext:
  • setNext(size_t increment)
    Note
    The first time it is called should return object in position 0,0 and set cursors to col:1 and row:0 (or row:1 if only 1 column), if limits are reached (col==getCols() && row==getRows()) this method can throw an BESInternalException or an SQLInternalException
    use notEnd to check end condition

Implemented in ODBCConnector, and SQLITESimpleConnector.
