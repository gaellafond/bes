Detailed Description

template<size_t sz>
class matched< sz >

This is a structure to keep result of the match function. The struct matched is sized on the maximum number of groups to match into a row but it only stores (into the match string array), the matched string. To know what groups are stored into the match array you can use the bitset<sz> map which has set to true only bit corresponding to the stored group.
