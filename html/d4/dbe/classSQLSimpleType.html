Detailed Description

template<class SQL_TYPE, class ODBC_TYPE = void, class DAP_TYPE = libdap::BaseType, class OUT = void>
class SQLSimpleType< SQL_TYPE, ODBC_TYPE, DAP_TYPE, OUT >

This is a implementation of a SQLBaseType abstract class This is useful when DAP_TYPE could be statically cast to OUT* which is written to the 'buf' variable through the set_val() method
WARNING!!! this template MUST be instantiate using a simple BaseType class like (Int16, Int32, Float32 ...) as DAP_TYPE
NOTE: if reuse is set to false returned cast pointer deletion will be performed.
