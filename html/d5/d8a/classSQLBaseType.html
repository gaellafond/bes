Detailed Description

template<class SQL_TYPE, class ODBC_TYPE = void, class DAP_TYPE = libdap::BaseType, class OUT = void>
class SQLBaseType< SQL_TYPE, ODBC_TYPE, DAP_TYPE, OUT >

This class is used to inherit dap components and to provide casting functionality to types. Clone if CLONE flag is set to 1.

See also
SQLCastAction. It is extended by:
  • SQLSimpleType
