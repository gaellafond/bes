Detailed Description

The SQLDataHandlerInterface is used as interface to pass container informations and usual object from BESDataHandlerInterface such as the BESResponseObject.

Note
: This class not inherit from BESDataHandlerInterface but own a reference to it and add a SQLContainer pointer loaded using the SQLContainerFactory::getContainer(dhi).
