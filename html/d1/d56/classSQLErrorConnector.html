Detailed Description

template<class ERROR_TYPE, class MSG_TYPE>
class SQLErrorConnector< ERROR_TYPE, MSG_TYPE >

This is the error part of a SQLConnector.

This interface should be implemented if you want to use a complete SQLConnector which uses the DynamiErrorManager part of the SQLHandler. Using DEM you'll be able to use standard actions provided by the platform and or add your own.
