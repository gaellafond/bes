Detailed Description

template<class K, class V, class T1 = std::shared_ptr<K>, class T2 = std::shared_ptr<V>>
class smart::SmartMap< K, V, T1, T2 >

Map to store object using smart pointers. Note that this class uses smart pointers for key and values. If you want to use smart management only for values, use SmartValueMap.

CLONE is set to 1 smart::Clone
