Detailed Description

template<class ERROR_TYPE, class ARGS_TYPE, class OUT = void>
class SQLCloseAction< ERROR_TYPE, ARGS_TYPE, OUT >

Implements the close action for the connector.
'ON_EVENT' actions are performed using the SQLErrorManager and the passed ErrorFactory. Set this variable to NULL to skip this phase.
You can also use the static method close(...) to make manual call.
NOTE: We haven't fixed the ARGS_TYPE and OUT types since they are unused. Set those parameter accordingly with the ActionList you want to use as container.
